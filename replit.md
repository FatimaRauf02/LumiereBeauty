## Overview

A complete luxury skincare/beauty ecommerce platform built on a pnpm monorepo. Dark theme with champagne gold accents.

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **API framework**: Express 5
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4`), `drizzle-zod`
- **API codegen**: Orval (from OpenAPI spec)
- **Build**: esbuild (CJS bundle)
- **Frontend**: React + Vite, wouter, @tanstack/react-query, framer-motion, recharts
- **AI**: Anthropic Claude (via Replit AI Integrations) for chatbot + quiz recommendations
- **Auth**: JWT (accessToken in localStorage, refreshToken in httpOnly cookie), bcryptjs

## Key Commands

- `pnpm run typecheck` ‚Äî full typecheck across all packages
- `pnpm run build` ‚Äî typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` ‚Äî regenerate API hooks and Zod schemas from OpenAPI spec

## Auth Flow

1. User logs in ‚Üí `POST /api/auth/login` returns `{ accessToken, user }`
2. `setAuthTokenGetter` in `use-auth.tsx` injects Bearer token into all API calls via custom-fetch
3. Token stored in `localStorage` key `lumiere_access_token`
4. Cart uses anonymous sessions via `x-session-id` header (UUID in `lumiere_cart_session`)
